Cute news是一个功能强大，但十分简单的新闻管理系统。它使用文件而不是数据库来保存新闻内容。它支持
 评论以及根据月份组织的归档文件。
该脚本是开放源码，你可以任意使用这个代码。但如果有比较大的更新，请
 联系我们。

 

 安装CuteNews

 

安装CuteNews十分简单，按照下面几步，应该不会有什么问题：

 1) 在你的服务器上创建一个目录 ( 比如 cutenews )
 2) 将cutenews.1.3.zip的内容上载在你在第一步中创建的目录
 3) 将第一步创建的目录及所有文件和子目录的属性改成777 
 4) 用浏览起访问http://yoursite.com/folder_from_step1/index.php
 5) 在安装CuteNews后，建议删除从服务器上 ./inc/install.mod  

 

 让你的页面显示新闻...

 

你可以通过在你的页面中加入简单的PHP代码，让你的页面显示新闻、大字标题以及归档。 
包含新闻的页面的后缀必须是 .php

简介

在显示news/headlines/archives时，你可以使用$number = x; 来限制每一页显示的最大数目到x条新闻。如果你使用 $number ， "下一页>>" 用来显示剩下的新闻。 

如果你想显示来自某一分类的news/headlines/archives，那你可以使用$category = ID; 其中，ID是文章分类的id，你可以选择不止一个分类：$category = ID1,ID2,ID3;

如果你想在同一页上显示新闻和大字标题，当选择了一个大字标题后，会在该处显示新闻的话，你必须在包含大字标题之前，使用$static_headlines = TRUE; 。
 

显示最近新闻

 

<?PHP
 include("path/to/show_news.php");
?>

 
 

显示大字标题

  <?PHP
 include("path/to/show_headlines.php");
?> 		 
 

显示归档

  <?PHP
 include("path/to/show_archives.php");
?> 		 
 

显示最近 5 条新闻

  <?PHP
 $number = 5;
 include("path/to/show_news.php");
?> 		 
 

显示最近 8 条大字标题 

  <?PHP
 $number = 8;
 include("path/to/show_headlines.php");
?> 		 
 

在同一页上显示所有的大字标题和最近新闻 

 

<?PHP
 $static_headlines = TRUE;
 include("path/to/show_headlines.php");

 $number = 1;
 include("path/to/show_news.php");
?>

 
 

备注：

  • 在上面所有的例子中，你必须将path/to/...替换成你的CuteNews的安装目录
    并且不要使用URL，而是目录
    错误: include("http://site.com/cutenews/show_news.php);
    正确: include("cutenews/show_news.php);                          
  • 在大多数情况下，你要把新闻和大字标题包括在同一页面，必须使用$static_headlines = TRUE; 。
  • 如果有任何问题，你可以到我们的服务论坛来，我们将很高兴能帮助你。
