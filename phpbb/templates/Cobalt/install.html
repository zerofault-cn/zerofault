Template Information
 

Thank you for installing Cobalt 2.0. This is the second incarnation of the hugely popular Cobalt template for phpBB 2. This new version improves on the old one with an improved look and feel as well as fixes for a number of problems and bugs.

Before you start using this template please read this whole document as it contains important information. It's important that you understand the license as you by installing Cobalt are legally bound to it.

With that in mind I hope you'll have much fun with this template and that it'll make running your own Internet community even more fun.

- Jakob Persson, 7th of January 2004

Installation
 
Installing the template should be fairly easy. I suggest you read this howto on installing templates. If you experience problems during installation, post at the phpBB support forums. Include a detailed description of the problem. Remember, the key to getting a good answer is asking a good question.
 
 
Blank Logo and Extras
 
In order to make it easier for you customize the forums with your own logotype I've attached a blank logo. Right-click here and select "Save as..." to save it on your hard drive. It's a lossless PNG file and can be edited with most common graphics editing applications such as Photoshop and Paint Shop Pro.

Template files for popular MODs
My friend Rivers has modified a few popular MODs to work with Cobalt, these include Acyd Burn's file attachment, the advanced quick reply, shop, points, lottery, bank and Smartor's Photo Album. You can download these files here. Do not regard the code as in any way complete, rather as a shortcut and a way to save time. Installing a MOD takes time and manual work and you should always remember to backup your files before attempting to modify your phpBB installation.
Please note that I am not responsible for any of the files that are released by Rivers and hosted at his site. Please refer to him if you have questions.

I will soon make files available at my website, if you're interested make sure to check it regularly.
